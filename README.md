# wordpress-starter-theme
Wordpress theme using the Bootstrap Starter Theme.  This is my first attempt at creating a custom theme in Wordpress.

## Setup
Download the .zip and install in your theme menu in Wordpress

## Technology Used
* PHP
* Wordpress
* Bootstrap
